# The Long Retreat

**A launch-candidate iPhone survival command game about leading a civilian convoy through a retreat that cannot be rewound.**

[Visit the official site](https://rzkowski.com/projects/the-long-retreat/)

## How It Works

You command the UNS Defiance across a 24-jump retreat, choosing routes, assigning ship support, and answering distress calls as fuel, pressure, and losses compound. The run log is append-only: choices cannot be rewound, and the Memorial keeps the names of the people command could not save.
